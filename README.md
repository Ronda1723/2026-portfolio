# rounaklenka.dev

Personal portfolio. Single-page static site — no build step.

## Run locally
```bash
python3 -m http.server 5173
# → http://localhost:5173/
```

## Deploy
Drop on Vercel / Netlify / GitHub Pages. No config needed.

## Stack
HTML + CSS + a tiny IntersectionObserver. Inter + Instrument Serif via Google Fonts.
