# images/

Drop image files here (jpg, png, webp). Reference them from `index.html` like:

```html
<img src="images/your-file.jpg" alt="description">
```

See the main [README](../README.md#how-to-add-images) for which sections accept images and the exact tag to paste.

Recommended sizes:
- Hero grid cells (Section 01): ~600×800 px (3:4)
- Project card media (Section 03): ~1200×600 px (2:1)
- Off-the-clock tiles (Section 06): ~800×600 px (4:3)

Keep files under ~300 KB each — the site has no build step / image optimization.
